# Changelog SwiftLogix

Semua perubahan penting pada proyek ini akan didokumentasikan di sini.

Format mengikuti [Keep a Changelog](https://keepachangelog.com/id/1.0.0/).

## [1.0.0] - 2025-06-14

### Ditambahkan
- Website frontend lengkap (Beranda, Tentang, Layanan, Tracking, Blog, Kontak)
- Panel admin dengan 8 modul CRUD
- Sistem tracking pengiriman real-time dengan riwayat status
- Manajemen konten blog dengan editor teks
- Manajemen testimoni pelanggan
- Manajemen anggota tim perusahaan
- Pengaturan situs dinamis (nama, kontak, media sosial)
- Autentikasi admin via Laravel Breeze
- Upload gambar untuk layanan, blog, testimoni, tim
- 30 unit & feature tests
- Dokumentasi lengkap (README, DATABASE, DEPLOYMENT, CONTRIBUTING)
- Seeder dengan data dummy realistis
- CSS custom dengan tema navy + oranye
- Animasi AOS (Animate on Scroll)
- Responsive design (mobile-friendly)

### Teknologi
- Laravel 13 + PHP 8.3
- MySQL 8
- Bootstrap 5.3
- Font Awesome 6
- AOS Animation Library
- Google Fonts (Inter + Poppins)
